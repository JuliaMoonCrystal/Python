# Jogo de Pedra, Papel e Tesoura

Um jogo simples de Pedra, Papel e Tesoura (JoKenPo) desenvolvido em Python.

## Como Jogar

1. Certifique-se de ter o Python instalado.
2. Navegue até esta pasta.
3. Execute o comando:
   ```bash
   python main.py
   ```

## Funcionalidades
- Placar em tempo real.
- Interface amigável no terminal.
- Animação simples de "Jo-Ken-Po".
